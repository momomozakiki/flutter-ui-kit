# Starter templates for consuming apps

The files in this folder are **starter templates** intended to be copied *out* of this repo into a brand-new consuming Flutter app and then adapted to that app's needs. They are **not part of the kit's own build** — editing them here does not change kit behavior, and their presence here is purely for reference and sharing.

Any `{{PLACEHOLDER}}` token must be filled in before use. See `ONBOARDING.md` (in the kit's root) for step-by-step setup instructions.
